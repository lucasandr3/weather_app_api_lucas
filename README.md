# Weather App Api Lucas

## Como executar o projeto

### Clonar o projeto

```
git@github.com:lucasandr3/weather_app_api_lucas.git
```

### Instalar as dependências

```
composer install
```

### Copiar o arquivo .env

```
cp .env-example .env
```

### Para rodar o servidor

```
php artisan serve
```

### Documentação swagger

/api/documentacao

### Requerimentos

- PHP 8.4
